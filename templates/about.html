What is ReciPy?

ReciPy is a web app that uses Python to find recipes for your ingredients!

  • Runs on Flask server
  • Uses Edamam API
  • Renders HTML with Jinja

Why this website?

For the Future Ready Talent Project and my vegan friends!

This web app was built for my future ready talent project where I wanted to combine the web development knowledge. Thanks to the amazing instructors and fellow students at Future Ready Talent community, I learnt a lot to be able to make this web app!
